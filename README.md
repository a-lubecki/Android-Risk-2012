# AndroidRisk-2012

AndroidRisk is a multiplayer game I made on Android in Master school.

The rules are the same as the Risk board game but the board map is the France map splitted by regions.

Here is the concept:
- you and your friends, create an account in the app, they're saved on the server
- you create a new game session and it's created on the server
- your friends join it
- the France map appear and you can start the game session as if you play the Risk game normally
- you can then play at your rythm: when you make a move, it's sent to the server, the next player is notified with a push notification
- when it's your turn, you can see the history of the session
